# Routeva 0.7.0

Routeva теперь работает без Happ, Amnezia и других VPN-клиентов на компьютере. Расширение подключается к вашему Routeva Gateway, а Gateway самостоятельно поднимает узлы из подписки через Mihomo.

## Что нового

- добавление подписки одной HTTPS-ссылкой с автоматическим импортом и обновлением;
- узлы VLESS, VMess, Trojan, Shadowsocks, Hysteria 2, TUIC, WireGuard и AmneziaWG отображаются как отдельные карточки;
- переключение узла перелистыванием карточки в popup;
- два режима: только сайты из списка или весь интернет;
- автоматический перенос ранее добавленной подписки на Gateway;
- флаг, страна и внешний IP активного узла после проверки;
- защищённый Gateway API, который не возвращает ссылку подписки и секреты расширению;
- исправлен сбой popup при нажатии до завершения загрузки состояния;
- подготовлены отдельная обычная и Chrome Web Store сборки.

## Установка

Для Brave распакуйте `routeva-extension.zip`, откройте `brave://extensions`, включите режим разработчика и выберите «Загрузить распакованное расширение».

Gateway должен быть установлен на вашем сервере по инструкции из `gateway/README.md`. Доступность конкретного узла зависит от провайдера подписки: Routeva поддерживает перечисленные протоколы, но не может восстановить недоступный или просроченный узел.
