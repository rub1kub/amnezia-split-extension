# Карточка Chrome Web Store

## Название

Routeva — VPN и прокси

## Краткое описание

Нужные сайты или весь интернет через HTTP, HTTPS или SOCKS-прокси. Подписки и несколько серверов.

## Подробное описание

Routeva направляет через ваш прокси выбранные сайты или, по вашему выбору, весь интернет. Исключения всегда можно оставить на обычном подключении.

Что внутри:

- HTTP, HTTPS, SOCKS4 и SOCKS5;
- импорт персональных HTTPS-ссылок подписок;
- распознавание VLESS, VMess, Trojan, Shadowsocks, Hysteria 2, TUIC, WireGuard и AmneziaWG;
- несколько серверов с переключением перелистыванием;
- два режима: по списку сайтов или весь интернет;
- страна и публичный IP активного выхода;
- готовый обновляемый список доменов для России;
- ручные сайты и исключения;
- локальное хранение настроек без аналитики и рекламы.

Важно: Chrome напрямую подключается только к HTTP/HTTPS/SOCKS. Узлы VLESS/VMess/Trojan и другие туннельные протоколы можно использовать через локальный SOCKS5-порт Happ, Amnezia или другого клиента.

## Категория и язык

- Категория: Tools / Инструменты
- Основной язык: русский

## Единая цель

Направлять выбранные пользователем домены или весь интернет через выбранный пользователем прокси с поддержкой явных исключений.
