# Инструкция для ревьюера Chrome Web Store

1. Установите расширение и откройте его настройки.
2. Добавьте доступный тестовый HTTP/HTTPS/SOCKS-прокси либо публичный прокси без авторизации.
3. Нажмите «Сохранить и проверить». После успешной проверки появятся страна и IP выхода.
4. Откройте любой HTTPS-сайт, нажмите значок расширения и включите «Текущий сайт».
5. Перелистните карточку, чтобы выбрать другой настроенный сервер.
6. Для проверки подписки настройте совместимый Routeva Gateway и вставьте контролируемую ревьюером HTTPS-ссылку Clash-подписки. Узлы появятся в списке и в карточках popup без локального VPN-клиента.

Расширение не содержит тестовых паролей или секретных подписок. Удалённые списки и подписки являются данными, исполняемый код с сети не загружается.
