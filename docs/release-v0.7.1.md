# Routeva 0.7.1

- Добавлен быстрый поиск узлов по названию, стране и протоколу.
- Убраны лишний флаг, уведомление после перелистывания и рывок карточки при открытии.
- Проверка IP теперь сбрасывается и запускается заново после переключения Gateway.
- Настройки показывают профиль подключения для редактирования, а узлы выбираются в окне расширения.
- Убраны повторяющиеся технические пояснения о Gateway.

После обновления распакованного расширения нажмите «Обновить» на `brave://extensions`.
